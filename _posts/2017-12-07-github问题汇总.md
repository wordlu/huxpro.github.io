## git命令：
 - #### 1.更新本地代码：
 	>          git fetch --all
	>          git reset -- hard origin/master
 - #### 2.提交本地代码到代码库：
 	>          git status //查看更改的代码
 	>          git add . //追踪所有代码 [或者使用 git add -u 只添加修改的文件]
 	>          git commit -m "说明" 
 	>          git push
