## git命令：

- #### 1.克隆项目到本地
	
```
		git clone [url]	
		git config --global user.name "name"
		git config --global user.email "email"
		git config --list	//查看所有配置信息

```

- #### 1.更新本地代码：

```
		git fetch --all
	    git reset -- hard origin/master
```

- #### 2.提交本地代码到代码库：

```	
 	    git status 				//查看更改的代码
 	    git add . 				//追踪所有代码，提交到暂存区 
 	    git commit -m "说明" 	//添加到版本库
 	    git push
```
